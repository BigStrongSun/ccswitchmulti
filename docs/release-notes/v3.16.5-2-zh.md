# CCSwitchMulti v3.16.5-2

## 修复

- 修复旧版 Codex MultiRouter 内联 OAuth 路由刷新后仍停留在旧模型目录的问题。
- 旧路由现在会按 OAuth 账号安全迁移到对应的 OpenAI Official provider；默认账号优先绑定稳定的 `codex-official`，多账号不会静默串号。
- 官方 OAuth 模型刷新后会同步更新旧路由的精确模型、模型映射与聚合目录，`gpt-5.6-luna`、`gpt-5.6-sol`、`gpt-5.6-terra` 可进入多路路由模型列表。

## 说明

- 子 Agent 前五候选继续保留用户已有排序；新增 5.6 模型会进入“路由/全部”候选池，不会擅自覆盖用户手工选择。
