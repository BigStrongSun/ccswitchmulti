# CCSwitchMulti v3.16.5-12

本版修复部分用户启动时出现“应用界面加载失败”并显示 `Cannot read properties of null (reading 'settingsConfig')` 的问题。

## Provider 配置兼容

- 修复历史导入、异常同步或手工数据库修改留下 `settings_config = null`、数组、标量或损坏 JSON 时，启动首屏崩溃的问题。
- 数据库读取 provider 配置时，非对象内容会安全规范化为 `{}`；有效对象保持不变。
- provider 保存和局部配置更新也会写入对象配置，避免坏数据再次落库。
- 已有用户无需手动编辑 SQLite 数据库；读取旧记录时会自动恢复为可打开的空配置。

## 验证

- Rust 格式检查与 `cargo check --lib` 通过。
- 数据库测试组 40/40 通过，覆盖 provider 列表读取、按 ID 读取和写回非对象配置。
