# CCSwitchMulti v3.16.5-11

本版修复 MultiRouter 在同一 Codex 会话中切换不同上下文窗口模型时，仍沿用前一模型全局上下文限制的问题。

## 按当前模型恢复上下文窗口

- MultiRouter 选路开启后，不再写入会覆盖所有模型的 `model_context_window` 和 `model_auto_compact_token_limit` 全局配置。
- 在同一会话从短上下文模型切至长上下文模型时，Codex 可读取当前目标模型的完整上下文窗口，不再被前一短模型限制。
- 从长上下文模型切回短上下文模型时，Codex 会按当前短模型的窗口恢复其官方的发送前压缩保护，避免把超长历史直接发送给短窗口模型。
- 单模型 Provider 保持原有配置行为，用户设置的全局上下文覆盖不会被此修复移除。
